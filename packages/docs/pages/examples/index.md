# Example projects using *Jadis*

Currently, the repository includes two examples:

## A Todo List

A simple demonstration of how to build a todo list using *Jadis*. It showcases component structure, event handling, and reactive updates, all with clean and type-safe code.

You can find the full source code [in the repository](https://github.com/bioleyl/jadis-examples/tree/main/packages/todo-list) and try it online [on this page](https://bioleyl.github.io/jadis-examples/todo-list/).

## A Minesweeper

A fun and playable project full of nostalgia...

You can find the full source code [in the repository](https://github.com/bioleyl/jadis-examples/tree/main/packages/minesweeper) and you can play it online [on this page](https://bioleyl.github.io/jadis-examples/minesweeper/).
